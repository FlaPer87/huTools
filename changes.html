Changes¶

  • 0.60p1: huTools.http implements authentication

  • 0.60: huTools.http API and function-based pyJasper Interface. Dependencie Reduction

    This version introduces a HTTP-API menat to allow easy use on pure Python and AppEngine plattforms. All other huTools modules should reley on them instead of httplib2 and others.

    pyjasper now comes with an non Object Oriented interface and also strips timestamps from generated PDFs to ensure the same input generates EXACTLY the same output.

    use hujson instead of simplejson, alsp big cleanup, github-tools/paver for documentation, better documentation at http://hudora.github.com/huTools/ , hujson, rework of the structured module, Python 2.5, Google AppEngine compabitility (also in hujson), reduced dependencies, http - a simple http client, unified credential configuration for postmark and jasper, support for signing in pyJasper,
