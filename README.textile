h1. huTools

huTools is a small collection of helpful tools we use.

* calendar - date based calculations and format conversions
 * calendar.formats - output and parsing of different date formats
 * calendar.workdays - date based calculations and format conversions
* checksummming - calculate various checksums
* decorators - various helpfull decorators
* luids - locally unique user-ids
* robusttypecasts - type conversionts that seldom fail
* printing - access printers
* ReReadingConfigParser -  ConfigParser which detects changes in config files
* unicode - real-world unicode handling
* world - country codes
* huLint - Code Quality Checking
