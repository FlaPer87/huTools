Standards / Protocols¶

We offer a set of guidelines how to structure Data for certain applications.

  • AddressProtocol for structured exchange of (postal) Adresses.
  • LieferungProtocol for delivery notes, packing orders etc.
  • VerySimpleOrderProtocol for purchase orders
  • VerySimpleStatusProtocol for track & trace information
  • Artikelstammdaten for basic product data
  • Messaging/Transport for transporting over messaging system
  • Messaging IC/WMS for messaging between an ERV System and an Warehouse Managment System
  • Messaging WMS=Logos Implementation Notes for the Warehouse Managment System “Logos”.
